    环境

    系统 : Windows 10

    NodeJS : 14.21.3

    Git : 2.3.8

    一、概述

    Hexo 中默认使用的主题是 landscape ,但是这个主题不便于博客的搜索和目录查看，这里我选择了 3-hexo 主题，下面介绍如何在Hexo中安装和使用该主题.

    如果想选择其它个性化主题，可从Hexo官网查找下载。

    Hexo主题：主题

    二、安装

    安装步骤参考 3-hexo 主题作者的安装步骤：安装步骤

    1. 下载主题

    windows打开 cmd 或者 Git Bash 进入hexo初始化博客文件夹下,执行如下命令

    git clone https://github.com/yelog/hexo-theme-3-hexo.git themes/3-hexo

    拉取成功，主题文件会被下载到 themes 文件夹下。

    2. 修改 _config.yml

    修改根路径(hexo博客文件夹所在路径)下的 _config.yml 文件配置，切换成刚下载的 3-hexo 主题。

    theme: 3-hexo
highlight:
  enable: false #关闭hexo渲染高亮，使用主题代码块高亮

    theme 和 highlight 这个两个配置在文件中都已存在，使用 Ctrl + F 找到后，修改保存！

    3. 安装wordcount插件

    使用npm安装字数统计插件，因为 3-hexo 这个主题使用了这个插件，所以必须装。

    npm i hexo-wordcount --save

    4. 修改主题配置文件

    位置：博客根路径\themes\3-hexo_config.yml 

    word_count: true

    5. 验证主题安装

    启动hexo，查看主题是否安装成功

    # 清空缓存
hexo clean
# 生成静态页面
hexo g
# 本地启动部署
hexo server
